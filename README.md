# 🎬 Movie Recommendation System

A simple Movie Recommendation System built using Python and Machine Learning techniques.

This project recommends movies based on movie genres using:

- TF-IDF Vectorization
- Cosine Similarity
- Content-Based Filtering

---

# 📌 Features

✅ Recommend movies similar to a selected movie  
✅ Uses machine learning concepts  
✅ Fast and lightweight  
✅ Simple Streamlit web interface  
✅ Beginner-friendly ML project

---

# 🛠 Technologies Used

- Python
- Pandas
- Scikit-learn
- Streamlit
